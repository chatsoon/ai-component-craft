## 🎯 使用指南

### 安装使用

#### 方式1: CLI工具
```bash
# 安装
npm install -g ai-component-craft

# 使用
ai-component-craft generate
```

#### 方式2: API调用
```typescript
import { AIGenerator } from 'ai-component-craft';

const generator = new AIGenerator('your-api-key');
const component = await generator.generateComponent({
  description: '创建一个按钮组件',
  framework: 'react',
  typescript: true
});
```

#### 方式3: VS Code扩展
1. 在VS Code中安装扩展
2. 右键点击文件夹
3. 选择 "生成AI组件"
4. 按向导完成生成
